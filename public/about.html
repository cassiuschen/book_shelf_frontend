假期回到宿舍，带来了好多书，突然发现整个书架快要放不下了，于是便萌发了把书借出去的想法。既然要借阅，便要有个地方管理书单、管理图书状态，照传统，便要写个网站。可毕竟没什么人想要为了找我借书而打开电脑登陆一个网站，于是用手机借就变得尤其的重要，

有了上述的需求，便使用了 IONIC 这个流行框架，本身 IONIC Framework 是搭配 PhoneGAP 做移动端App的框架，那么这次我只用它 Web View 的部分完成了整个页面的设计。最终，我使用了 Angular.js / Gulp.js / Bower.js 作为三个主要的开发工具，并在3天的业余时间完成了这套前端。后台使用 Ruby on Rails 配合 MongoDB 完成数据储存以及为该借书前端提供API。

该项目在 GitCafe 和 Github 上开源。

下一步，会逐渐开放电子书等电子资源的预览与下载。目前纸质书依靠 ISBN 编码通过豆瓣获取简介等元信息，电子资源则需自行编目，还需要时间。
